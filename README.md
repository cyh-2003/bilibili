# 使用
```bash
node index.js
```
# 参数

-flv 获取FLV/MP4格式视频

```bash
node -flv
```

-qn 指定视频清晰度(仅FLV/MP4格式可用)

```bash
node -flv -qn 64
```


| 值  | 含义       |
|:---|:---------|
| 6  | 240P 极速  |
| 16 | 360P 流畅	 |
| 32 | 480P 清晰 |
|64| 720P 高清  |
> 根据具体视频指定

-lesson 获取课程视频
